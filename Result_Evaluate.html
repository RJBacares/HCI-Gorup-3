Key Findings
  • The system successfully recommends jobs based on user skills.
  • Location-based filtering helps users find nearby opportunities.
  • Some mismatches occur due to incomplete user profiles.
  • System speed affects real-time job matching.
Discussion

The results indicate that the job matching feature is functional and beneficial for users in Bicol. However, improvements in algorithm accuracy and data completeness are necessary to ensure better job recommendations. Enhancing system responsiveness will also improve user experience.

Recommendations
  • Improve matching algorithm for higher accuracy
  • Require complete user profiles before matching
  • Enhance filtering by skills and location
  • Optimize system for faster results
  • Integrate real-time job updates
