Bicol Job Working System

Results, Discussion, and Recommendations

Summary of Results

The usability evaluation of the Bicol Job Working System shows that users were able to understand basic system functions, but experienced difficulties in navigation, clarity of job information, and system responsiveness. The system is functional but requires improvements to enhance user experience and efficiency.

  • Overall Usability: Moderate
  • User Satisfaction: Average
  • Main Issue: Navigation & Clarity
  • Status: Functional but needs improvement

Results

Key Findings

  • Users had difficulty navigating job listings on mobile devices.
  • Some job posts lacked complete descriptions.
  • Button labels were unclear in some sections.
  • System loading speed affected user experience.

Discussion

The evaluation indicates that the system does not fully comply with UI/UX principles such as clarity, consistency, and ease of navigation. Users struggled to interact efficiently with the system, especially when accessing job listings. These issues suggest that improvements in design and performance are necessary to enhance usability and user satisfaction.

Recommendations

  • Improve navigation with a more user-friendly layout.
  • Ensure all job listings have complete and clear details.
  • Use consistent and descriptive button labels.
  • Optimize system performance for faster loading.
  • Enhance UI design for better readability and structure.
  • Conduct further usability testing after improvements.
